# أثر التدريب العملي في تعزيز الكفاءات الرقمية

عرض تقديمي حديث مُعاد بناؤه باستخدام Next.js 14 و Tailwind CSS و Framer Motion

## 🚀 المميزات

- **تصميم عصري** باستخدام Tailwind CSS
- **حركات سلسة** مع Framer Motion
- **رسوم بيانية تفاعلية** باستخدام Chart.js
- **متجاوب بالكامل** مع جميع الأجهزة
- **تنقل سهل** بين الشرائح
- **مؤشرات جانبية** مع أسماء الشرائح
- **لوحة تقدم** تفاعلية
- **دعم كامل للغة العربية**

## 📋 المتطلبات

- Node.js 18 أو أحدث
- npm أو yarn

## 🛠️ التثبيت والتشغيل

1. **تثبيت الاعتماديات:**
   ```bash
   npm install
   # أو
   yarn install
   ```

2. **تشغيل خادم التطوير:**
   ```bash
   npm run dev
   # أو
   yarn dev
   ```

3. **فتح المتصفح على:**
   ```
   http://localhost:3000
   ```

## 📦 البناء للنشر

```bash
npm run build
npm start
```

## 🔧 التخصيص

### إضافة شريحة جديدة
1. افتح `app/page.tsx`
2. أضف الشريحة الجديدة في مصفوفة `slides`
3. أنشئ المكون في `components/SlideComponents.tsx`
4. أضف الشرط في `AnimatePresence`

### تعديل الألوان
عدل الألوان في `tailwind.config.js`:
```javascript
theme: {
  extend: {
    colors: {
      primary: '#2563eb',
      secondary: '#7c3aed',
      accent: '#f59e0b',
      // ...
    }
  }
}
```

### إضافة رسوم بيانية جديدة
1. أضف البيانات في `sampleData` في `SlideComponents.tsx`
2. استخدم `ChartComponent` مع النوع المناسب
